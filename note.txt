CSS Grid System
---------------
- element becomes a grid container when its display property is set to grid or inline-grid.

- The column-gap property sets the gap between the columns
Eg:column-gap: 2rem;

- The row-gap property sets the gap between the rows
Eg:row-gap:2rem;
